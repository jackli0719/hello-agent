## 6. 问题分级

按优先级从高到低：

| 级别 | 含义 | 处理时机 |
|---|---|---|
| **P0** | 影响功能 / 业务正确性 — 必须修 | 立刻 |
| **P1** | 影响使用 / 演示体验 — 应该修 | 本周内 |
| **P2** | 影响美观 / 易读 — 建议修 | 下个版本 |
| **P3** | 锦上添花 — 可选 | 看时间 |
